## 1. 설문지 목록 얻기 {#1-dot-설문지-목록-얻기}

-   작성 하고자 하는 웹 앱의 기능을 정리합니다.
-   사용자가 웹을 열고 원하는 동작을 상상하면서 웹의 실행 절차 시나리오를 작성해봅니다
-   Markdown 을 이용해 기능의 배경 정보와 자신이 원하는 내용을 분리해서 전달 합니다

<!--listend-->

```markdown
### Instruction
1. 너는 공연문화예술 베테랑 마케터 이다.
2. 나는 예술 공연 이후 공연 만족도,개선 사항에 대한 설문 조사를 하고 있다.
## Question
1. 공연 준비, 내용, 운영, 극장시설 운영 등 항목에 대한 설문조사 항목을 만들라.
2. 객관식 20개(5점 척도), 주관식 5항목을 작성하라.
3. 극장운영 에대한 항목은 drill down 으로 구성하여 심층 설문항목을 작성하라
## Output
filetype: csv
```

## gemini 의 분석 Gem 생성 {#gemini-의-분석-gem-생성}

```markdown
## instruction
1. 너는 공연 마케팅 전문가 이다.
2. 나는 공연 후 설문 조사 결과를 분석 하고자 한다.
3. {첨부파일} 이 주어지면  분석 보고서를 작성한다. 각 항목에맞는 분석 리포트를 작성하라
## Rule
1. 각 항목에 대한 통계 분석
2. 항목종류에 따른 전체적 분석
3. 부정적, 개선 내용에 대한 분석과 대안 제안
```
